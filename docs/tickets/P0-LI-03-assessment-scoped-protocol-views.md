# P0-LI-03 — Assessment-Scoped Protocol Views

## Purpose

Allow users to view protocols derived from a single assessment.

## In Scope

- Scoped protocol views per assessment.
- Provenance preserved in the scoped view.

## Out of Scope

- Consolidation logic.
- Export.

## Dependencies

- P0-LI-01 protocol identity and provenance.

## Acceptance Criteria

- Assessment A shows only its protocols.
- No leakage from other assessments.

## Manual Verification Steps

- Complete two assessments.
- Open each assessment’s protocol view and confirm isolation.
